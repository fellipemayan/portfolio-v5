# Changelog

Todas as mudanças relevantes deste projeto serão documentadas aqui.

## [1.0.0] - 2026-03-03

### Adicionado

- Estrutura inicial do projeto com Next.js
- Layout base e navegação
- Página inicial, sobre, contato e projetos
- Componentes principais: Header, Footer, CustomCursor, ProjectList, ProjectHeader, MotionWrapper, ContentRenderer, ContentSummary, Colophon
- Estrutura para listagem e detalhamento de projetos
- Estilização global e temas
- Hooks customizados (ex: useWeather)
- Configuração de arquivos estáticos e exemplos de projetos

As próximas versões trarão novos projetos e melhorias na experiência do usuário.
